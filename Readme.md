# QUIZZ APP WITH TCP
## HOW TO RUN IT ON LOCAL
### For server
Install the following

- Install make files

    `
    sudo apt install make
    `
- Install mysql-server

    `
    sudo apt install mysql-server
    `

    `
    sudo apt-get install libmysqlclient-dev
    `
- Run server

    `
    cd server
    `
    
    `
    make
    `
    
    `
    ./database
    `
    
    `
    ./server
    `

### For Clients
Run the client in Xampp
